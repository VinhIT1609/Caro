GIỚI THIỆU

Caro hay còn được biết đến với cái tên Gomoku hoặc Five in a Row. Mục tiêu của trò chơi là xếp được năm nước đi của mình tạo thành một hàng ngang, hàng dọc hoặc hàng chéo trước khi đối thủ làm được điều đó. Với sự kết hợp giữa chiến thuật và tư duy, Caro đã trở thành một trò chơi phổ biến và thú vị được nhiều người ưa thích. Ứng dụng trò chơi Caro 2 người chơi được xây dựng dựa các mục tiêu của trò chơi trên được viết bằng ngôn ngữ Python và sử dụng thư viện Socket cho phép kết nối trực tiếp giữa 2 người chơi với nhau thông qua mạng.

HƯỚNG DẪN CÀI ĐẶT

Bước 1: Tải xuống source-code từ GitHub.

Tải xuống GitHub thông qua đường dẫn sau:

https://github.com/DungIT2k2/Caro-Socket

Giải nén và thu được thư mục game Caro.

Bước 2: Cài đặt python với hệ điều hành tương thích.

Đối với Windows:

• Truy cập đường dẫn sau: https://www.python.org/downloads/

• Tải file .exe và tiến hành cài đặt trên máy tính.

Đối với Ubuntu:

• Mở Terminal và nhập lệnh

sudo apt install python3

• Tiếp theo tiến hành cài đặt pip.

sudo apt install -y python3-pip

• Nâng cấp pip và setuptools mới nhất bằng lệnh

pip3 install --upgrade pip

pip3 install --upgrade setuptool

Bước 3: Cài đặt các thư viện cần thiết.

Mở Terminal trong thư mục Caro, và tiến hành cài đặt thư viện

Đối với Windows:

• Sử dụng lệnh:

pip install -r requirements.txt

Đối với Ubuntu:

• Sử dụng lệnh:

pip3 install -r requirements.txt

• Cài đặt thư viện "tk":

pip3 install -r requirements.txt

Bước 4: Chạy ứng dụng.

Tiến hành chạy ứng dụng trên hệ điều hành tương thích.

Đối với Windows:

• Sử dụng lệnh:

python run.py

Đối với Ubuntu:

• Sử dụng lệnh:

python3 run.py
