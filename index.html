Introduce

Trò chơi Caro - một trò chơi với nguồn gốc từ trò chơi gomoku cổ điển, nổi tiếng với tên gọi "Five in a Row", là một trong những trò chơi chiến thuật thú vị và sâu sắc. Với lịch sử hơn một thế kỷ, Caro không chỉ là một trò chơi giải trí, mà còn là một bài toán chiến thuật được nhiều người yêu thích và tìm hiểu. Trò chơi Caro đòi hỏi người chơi phải có sự tư duy chiến thuật, khả năng dự đoán và phản ứng nhanh nhạy. Trên bàn cờ 15 X 15, hai người chơi sẽ thay phiên nhau đánh dấu X hoặc O trên các ô trống, với mục tiêu tạo thành một dãy liên tiếp gồm 5 dấu X hoặc O theo hàng ngang, cột dọc hoặc đường chéo.

Installation Guide

Bước 1: Tải xuống source-code từ GitHub.

Tải xuống GitHub thông qua đường dẫn sau:

https://github.com/DungIT2k2/Caro-Socket

Giải nén và thu được thư mục game Caro.

Bước 2: Cài đặt python với hệ điều hành tương thích.

Đối với Windows:

• Truy cập đường dẫn sau: https://www.python.org/downloads/

• Tải file .exe và tiến hành cài đặt trên máy tính.

Đối với Ubuntu:

• Mở Terminal và nhập lệnh

sudo apt install python3

• Tiếp theo tiến hành cài đặt pip.

sudo apt install -y python3-pip

• Nâng cấp pip và setuptools mới nhất bằng lệnh

pip3 install --upgrade pip

pip3 install --upgrade setuptool

Bước 3: Cài đặt các thư viện cần thiết.

Mở Terminal trong thư mục Caro, và tiến hành cài đặt thư viện

Đối với Windows:

• Sử dụng lệnh:

pip install -r requirements.txt

Đối với Ubuntu:

• Sử dụng lệnh:

pip3 install -r requirements.txt

• Cài đặt thư viện "tk":

pip3 install -r requirements.txt

Bước 4: Chạy ứng dụng.

Tiến hành chạy ứng dụng trên hệ điều hành tương thích.

Đối với Windows:

• Sử dụng lệnh:

python run.py

Đối với Ubuntu:

• Sử dụng lệnh:

python3 run.py
